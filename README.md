# Informatics

Variety of different problems from various competitions or practice. Majority of problems come from train.nzoi.org.nz.

Code include but are not limited to: binary search, bfs, dfs, djikstra, bellman-ford, floyd-warshall, floyd's tortoise and hare, topological sort, disjoint set union, fenwick trees, prim's, kruskal's, dynamic programming, segment trees (with and without lazy propagation), dinic's, convex hull.
